# Buffer 缓冲区

## 含义

在 NIO 中负责数据的存取，本质是数组，用于存储不同类型的数据。主要包括如下几类：

- ByteBuffer
- CharBuffer
- ShortBuffer
- IntBuffer
- LongBuffer
- FloatBuffer
- DoubleBuffer

实例化方式一致，即 `XXBuffer = XXBuffer.allocate();`
